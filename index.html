Unreal Engine 5.2 - Game Dev Skill Level II

An attempt to replicate Mass Effect Andromeda, on a manageable scale of a one-man team, but within a short span of time. I mean I was adoring this game in my previous project like a fangirl. Why not try to assimilate it?

Challenge Accepted!

Intention

To develop another small Unreal Engine 5.2 demo to demonstrate my improved Unreal Engine C++ skills over the months after my first project.

  • The complexity of this project will be 3 times that of my last project as there are way more features I will attempt at developing. The project will incorporate component classes, data tables, curves, TMaps, object pools, dialogue system, MetaHumans, etc. with better, cleaner code structure.
  • Since this is an RPG, it may have one questline.
  • There will be NPCs but they will be static.
  • I may be using a free generative AI tool to create voicelines.
